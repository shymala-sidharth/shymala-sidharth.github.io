Time that I was blocked on a simple problem

The problem was to add new elements to an array of object. I knew there is an easy way of doing it but it took a really long to get it to work. I tried my best to look at all the course resources and look at the documentation but I was still blocked. I finally found a stackoverflow page that discussed the same problem as mine (kind of) and it worked. how did you feel throughout the process? I learnt that problem solving is not a linear process and can take time and readjusting and rethinking several aspects of your code to get it to do what you want.

To be honest, this has not happened for me yet. I still find myself referring to the documentation and other materials for guidance to solve my problem. Because my brain hasn't fully grasped computer logic yet, I feel like I am fumbling though it. I know this is a process though and will happen more elegantly over time.

Problem Solving Techniques/Processes

Trying something: Very confident. I like giving it a go and see where it takes me.

Rubber Ducky Method: I kinda do this with my partner. I explain the problem to him and while doing so, I feel I see things better just by explaining the issue.

Reading the error message: Very helpful and I feel like I am getting better and reading it.

Console.log: I need to implement this more.
